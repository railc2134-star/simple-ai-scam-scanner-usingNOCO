# Noco DB Scanner

This project uses PyTorch to scan messages from a Noco DB instance for potential spam or scams. It utilizes the `requests` library to fetch data from the Noco DB API and `torch` for building and training a simple neural network model.

## Technologies Used
- Python 3.x
- PyTorch
- Requests

## How to Run
1. Clone the repository and navigate to the root directory.
2. Install the required dependencies using `pip install -r requirements.txt`.
3. Replace the placeholders in the code with your actual Noco DB API credentials and URL.
4. Run the script using `python toto.py`.
